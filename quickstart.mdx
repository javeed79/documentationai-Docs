## Prerequisites

<Callout kind="info">
You need a valid email address. No credit card is required to start.
</Callout>

## Create your workspace

<Steps>
  <Step title="Sign up" icon="user-plus">
    Go to the Linear sign-up page and create an account with your email. Verify your address to continue.
  </Step>
  <Step title="Name your workspace" icon="edit">
    Enter a workspace name such as `Acme Team`. Linear creates a default team inside the workspace automatically.
  </Step>
  <Step title="Invite teammates" icon="users">
    Add email addresses for your team members. They receive an invitation link and join instantly.
  </Step>
</Steps>

## File your first issue

<CodeGroup tabs="cURL,JavaScript">
```bash
curl -X POST https://api.example.com/issues \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"title":"Update landing page copy","team":"eng"}'
```

```javascript
const response = await fetch("https://api.example.com/issues", {
  method: "POST",
  headers: {
    Authorization: "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    title: "Update landing page copy",
    team: "eng",
  }),
});
```
</CodeGroup>

## Basic navigation

After the first issue appears in your inbox view, use the sidebar to switch between Inbox, My Issues, and All Issues. Press `C` to create a new issue from anywhere. Press `?` to open the keyboard shortcuts panel.
