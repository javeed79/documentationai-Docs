## Overview

Linear delivers a focused issue tracking experience built around speed and clarity. You organize work through cycles that align with your team's rhythm, projects that group related issues, and flexible views that surface exactly what matters. Keyboard-first navigation keeps your hands on the keys so you stay in flow.

## Cycles and Projects

Cycles replace rigid sprints with rolling two-week windows that automatically roll forward. You assign issues to the current cycle and watch progress update in real time.

Projects let you bundle issues that share a goal without forcing every task into a cycle.

## Integrations

Linear connects with the tools you already use. GitHub, Slack, and Figma sync status automatically so context stays in one place.

<CodeGroup tabs="CLI,API">
```bash
linear issue create --title "Fix login redirect" --project "Auth"
```

## Best Practices

Keep cycles short and focused. Use projects to capture outcomes rather than every task. Build views that answer recurring questions instead of creating one-off filters.
